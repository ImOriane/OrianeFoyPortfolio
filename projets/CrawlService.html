CrawlService

Voir le code Voir la documentation

Dans le cadre de mon stage dans l'entreprise Ekino, j’ai du développer une application web en Laravel permettant d’analyser les performances SEO et GEO d’un site à partir de son URL.

Le problème identifié était le manque d’outil centralisé et simple d’utilisation pour détecter rapidement les freins techniques impactant le référencement et la visibilité locale d’un site web. J’ai donc conçu une solution capable de crawler automatiquement un site, de collecter des métriques clés (structure des pages, balises, éléments SEO/GEO) et d’analyser ces données afin d’identifier les points d’amélioration.

Les résultats sont présentés dans un tableau de bord clair et structuré, offrant une vue globale des performances ainsi qu’un accès détaillé aux problématiques détectées pour chaque page.

Ce projet m’a permis de renforcer mes compétences en Laravel, gestion de bases de données, traitement automatisé de données et conception d’interfaces orientées analyse.
